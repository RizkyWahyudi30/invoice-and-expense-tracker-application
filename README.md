# Expense Tracker Application

ExpenseTracker adalaha aplikasi manajemen keuangan bisnis yang dirancang khusus untuk Freelancer, Pemilik UMKM ataupun Pembisnis. Aplikasi dibuat dengan tujuan untuk menangani beberapa masalah yang dialami user:

- Membuat dan Mengelola invoice kepada klien
- Memantau status pembayaran invoice secara real-time
- Mencatat dan mengkategorikan pengeluaran bisnis
- Melihat laporan keuangan bulanan yang diringkas
- Mendownload invoice dalam format PDF
